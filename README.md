# claude-skills

A collection of Claude Code skills I've created for software engineering tasks.

## Installation

Install with the [`skills`](https://www.npmjs.com/package/skills) CLI:

```sh
# project-level
npx skills add bradennss/claude-skills

# global
npx skills add bradennss/claude-skills -g
```

You can also install specific skills:

```sh
npx skills add bradennss/claude-skills --skill <skill-name>
```
